🚀 Retro Shooter Arcade
Um jogo retrô em pixel art inspirado nos clássicos de arcade, desenvolvido em HTML5 + JavaScript.
O jogador controla uma nave espacial que dispara automaticamente contra ondas de inimigos, com upgrades, power-ups e ranking de pontuação.

🎮 Características
Nave em estilo 8-bit retrô

Sistema de tiros com 4 níveis de upgrade

4 tipos diferentes de disparo:
🔸 Básico
🔸 Explosivo
🔸 Perfurante
🔸 Ricochete

Ondas progressivas de inimigos

Chefes desafiadores a cada fase

Ranking de pontuação salvo no navegador

Tela de Game Over estilizada com transição suave

Efeitos sonoros arcade (tiro, explosão, power-up, game over)

Fade-in/Fade-out entre menu, jogo e Game Over

Visual com efeitos neon e fundo espacial animado

🛠️ Tecnologias
HTML5 (Canvas API)

CSS3 (animações e estilo retrô)

JavaScript (lógica do jogo)

▶️ Como jogar
Abra o link do GitHub Pages: **[Retro Shooter Arcade](https://rayderluck.github.io/retro-shooter/)**

Digite seu nome no menu inicial.

Clique em START para começar.

Use o mouse para mover a nave (ou setas ↑ ↓ se preferir).

A nave dispara automaticamente — alinhe os tiros com os inimigos.

Colete power-ups para mudar o tipo de tiro ou aumentar o nível.

Sobreviva às ondas e derrote os chefes para subir no ranking!

📌 Objetivo
Destruir todas as ondas de inimigos, derrotar os chefes e alcançar a maior pontuação possível.
Mostre seu nome no TOP 5 Ranking Arcade!

🎵 Créditos
Sons retrô de explosão, tiro e power-up

Fonte Press Start 2P (Google Fonts)
