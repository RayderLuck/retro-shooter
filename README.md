# 🚀 Retro Shooter Arcade

Um jogo retrô em pixel art inspirado nos clássicos de arcade, desenvolvido em HTML5 + JavaScript.  
O jogador controla uma nave fixa na lateral da tela que dispara automaticamente contra ondas de inimigos.

## 🎮 Características
- Nave em estilo 8-bit
- Sistema de tiros com 4 níveis de upgrade
- 4 tipos diferentes de disparo (básico, explosivo, perfurante e ricochete)
- Ondas progressivas de inimigos
- Chefes desafiadores a cada fase

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript (Canvas API)

## ▶️ Como jogar
1. Abra o link do GitHub Pages: **[Retro Shooter Arcade](https://rayderluck.github.io/retro-shooter/)**
2. Use as setas ↑ e ↓ para mover a nave.
3. A nave dispara automaticamente — alinhe os tiros com os inimigos.
4. Colete power-ups para mudar o tipo de tiro ou aumentar o nível.

## 📌 Objetivo
Destruir todas as ondas de inimigos e alcançar a maior pontuação possível!
